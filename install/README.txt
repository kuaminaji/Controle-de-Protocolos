1. Execute o arquivo install.bat como administrador.
2. O MongoDB será instalado/configurado localmente.
3. O backend será iniciado automaticamente.
4. Acesse a aplicação pelo navegador: http://localhost:8000/
5. Login inicial: usuário admin, senha definida no código.
6. Para empacotar como .exe, utilize o PyInstaller conforme instruções no script.
7. Atalhos e .bat são criados automaticamente na área de trabalho.
